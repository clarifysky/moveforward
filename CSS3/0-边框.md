# 边框

- border-radius
- box-shadow
- border-image

## 圆角

```css
div {
    border: 2px solid;
    border-radius: 25px;
}
```

## 盒阴影

```css
div {
    box-shadow: 10px 10px 5px #888;
}
```

## 边界图片

div {
    border-image: url(example.png) 30 30 round;
    -webkit-border-image: url(example.png) 30 30 round;
    -o-border-image: url(example.png) 30 30 round;
}
